Features

  • QR-based manual entry tracking using a mobile application.
  • Real-time building occupancy updates stored and managed through Supabase.
  • AI-powered people counting using YOLOv8 and ByteTrack to automate entry/exit detection.
  • Multi-building support with a centralized backend connected to a live dashboard.
  • REST API integration enabling communication across mobile app, AI service, and database.

Technology and Implementation

MIT App Inventor: Used to develop the exhibition-deployed mobile app for QR-based entry scanning.

Node.js + Express: Backend API handling building states, logs, and AI data updates.

Supabase: Cloud database with RLS, real-time updates, triggers, and RPC functions.

Python + YOLOv8: AI people counting module with tracking, ROI detection, and video stream handling.
